# synthesizer-windows-console
Creating a simple sound synthesizer for Windows PC. I learnt this implementation and tips from the youtube channel @/javidx9

## Controls (Keys)
- C -> "Z"
- C# -> "S"
- D -> "X"
- D# -> "C"
- E -> "F"
- E# -> "V"
- F# -> "G"
- G -> "B"
- G# -> "N"
- A -> "J"
- A# -> "M"
- B -> "K"

## How to run
- Make sure you have the cpp compiler on your system
- Clone this repo
- Run the SoundSythesizer.cpp file
